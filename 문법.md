## 생성법 예제
```html
<div id="app">
    <h2>{{  message }}</h2>  <!-- ${message}  jsp el 표현식과 같다.-->
    </div>
    <script>
    let v = new Vue({
        el:"#app",    // el 은 엘리먼트 약자
        // data:{
        //     message: "Hello Vue!!!!"
        // }
        data(){
            return{
                message: "Hello Vue!!"
            }
        }
});
```
